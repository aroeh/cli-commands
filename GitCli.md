# Clone Repository

Exactly as it sounds.  Clones the specified repository to a workstation

`git clone <repo>`

```
git clone https://github.com/aroeh/cli-commands.git
```

> Retrieve changes

Gets the latest changes from the origin

`git pull`

```
git pull
```
